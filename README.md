**Interactive Recipe Card**

A simple and responsive HTML + CSS + JavaScript project that displays an interactive recipe card. Users can view ingredients, steps, and toggle sections dynamically.

**Features**

Clean and responsive UI

Interactive buttons using JavaScript

Simple recipe layout

**Technologies Used**

-HTML5

-CSS3

-JavaScript
